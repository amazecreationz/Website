We have started out to solve commoners' everyday problems with intelligent, well designed software. We believe a clear, simple and beautiful interface makes the best kind of software and thus, we rebuild our products regulary to adapt with the evolving technologies.
We use Google's state of the art cloud services for all our serving purposes. All our websites uses Firebase to deploy the web content to a global content-delivery network (CDN), giving the best performance even with busy traffic. Most of our server operations are run on Cloud Functions which again guarantee the best of performance.
